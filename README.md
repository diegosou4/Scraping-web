# Scraping For Uber Eats
## About
This is a progam for take latitude and longitude and return JSON file for use in GPS Manager or GPS Master (this app need Jailbreak). <br>
   
   
   ```bash
  git clone https://github.com/diegosou4/Scraping-web/ repo 
  cd repo
  sudo install npm
  cd src
  node index.js
  ```
